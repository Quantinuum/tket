contains a wasm op that could be added to a circuit.

This op stores in its signatures which bits are interacting as input and output with the call to which function of the wasm file

Parameters
_ntotal number bits it is interacting with
_ww_ntotal number bits it is interacting with
_width_i_parametervector of bits for each input i32
_width_o_parametervector of bits for each output i32
_func_namename of the function
_wasm_uiduid of the wasm file to be called
